# Building a Cybersecurity Company from West Africa

## The Beginning

In 2025, I was a pentester in Freetown, Sierra Leone, juggling 20 different tools for every engagement. Subfinder for subdomains. Nmap for ports. FFUF for fuzzing. Nuclei for vulnerabilities. SQLMap for injection. And a dozen more. Each tool had its own syntax, its own output format, its own quirks.

I thought: *"Why isn't there one platform that does all of this?"*

That question became TALOS.

## The Problem with Cybersecurity Tools

The cybersecurity industry has a fragmentation problem. There are incredible open-source tools available — but they don't work together. A pentester doing a full-scope assessment might use:

- 5 tools for reconnaissance
- 4 tools for enumeration
- 8 tools for vulnerability analysis
- 3 tools for exploitation
- Manual processes for reporting

Each tool outputs in a different format. None of them talk to each other. The pentester spends 60% of their time managing tools and 40% actually finding vulnerabilities.

TALOS flips that ratio.

## Building from Sierra Leone

People ask: "Why build a cybersecurity company from Sierra Leone?"

My answer: "Why not?"

The internet doesn't care where you're from. Code compiles the same in Freetown as it does in San Francisco. A zero-day exploit found from West Africa is just as dangerous as one found from Silicon Valley.

Building from Sierra Leone gave me advantages:

1. **Low cost of living** — I could afford to spend months building without venture capital
2. **Unique perspective** — I see security problems differently than someone in a major tech hub
3. **Hunger** — When you're from a place with few tech companies, you build your own

## The Technology

TALOS is built with:

- **Frontend:** Next.js 16, TypeScript, Tailwind CSS, shadcn/ui
- **Backend:** Python 3.13, FastAPI, SQLite
- **Infrastructure:** Docker, Nginx, Oracle Cloud
- **Tools:** 122 integrated security tools

The entire platform runs on Oracle Cloud's free tier — 4 ARM cores, 24GB RAM, 200GB storage. Zero infrastructure cost.

## What We've Built

Today, Zerodin has seven products:

1. **TALOS** — Universal Pentesting Platform (16 attack surfaces, 122 tools)
2. **SENTINEL** — Continuous Security Monitoring
3. **ODIN** — Threat Intelligence
4. **FORGE** — Automated Exploitation Engine
5. **NEXUS** — Cross-Platform Intelligence
6. **SLEIPNIR** — Mobile Security Platform
7. **HOPE** — Drone Delivery Platform

All built by one person. From Freetown. With no venture capital. No team. Just a laptop, Kali Linux, and a vision.

## The Mission

We exist to protect organizations against the unknown. The most dangerous threats are the ones nobody knows about — and we see them first.

Zerodin proves that world-class cybersecurity can come from anywhere. You don't need to be in Silicon Valley. You don't need millions in funding. You need a problem to solve and the determination to build the solution.
